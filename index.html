Sudoku Solver

Project Description:

Welcome to my Sudoku Solver project! This application is designed to solve Sudoku puzzles automatically. Simply enter the initial numbers into the Sudoku board and click the "Solve!" button to see the solution.

The solver uses a backtracking algorithm to explore all possible number combinations and find the correct solution. It starts by examining each empty cell and tries different numbers that comply with the Sudoku rules. If a number leads to a conflict, it backtracks and tries the next number until a valid solution is found.

Sudoku Rules:

  1. Each row must contain the numbers 1-9 with no repetitions.
  2. Each column must contain the numbers 1-9 with no repetitions.
  3. Each 3x3 box must contain the numbers 1-9 with no repetitions.
